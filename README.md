- Server : heroku
- Framework : spring boot 2.6.7, myBatis 2.2.2, bootstrap
- language : java8
- Tool : eclipse, mysqlWorkBench

## 주요 기능
- 스프링 시큐리티를 이용한 로그인
- 게시글 좋아요, 신고 기능
- 유저 신고, 쪽지 보내기 기능
- 내 게시글에 댓글, 쪽지를 받았을 시 알림 기능
- 관리자 기능 (게시글, 댓글 삭제, 신고당한 게시글, 유저 관리)
